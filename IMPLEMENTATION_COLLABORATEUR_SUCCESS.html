🔄 Architecture Technique

Séparation complète : Deux interfaces distinctes visuellement

Partage intelligent : Même base de données, permissions différentes

Admin Interface : Gère TOUS les collaborateurs └── Routes : /dashboard, /collaborateurs, /planning, etc. Collaborateur Interface : Gère SES propres données └── Routes : /collaborateur/dashboard, /collaborateur/*, etc. Services Partagés : ├── CollaborateursServiceV2 (admin : full access) └── CollaborateurSelfService (collaborateur : own data only) Database : Firebase Firestore + RTDB ├── tenants/{tenantId}/collaborateurs/{id} (shared) └── tenants/{tenantId}/disponibilites/{id} (shared)
