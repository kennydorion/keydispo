🎨 Test Synchronisation Couleurs Avatar

📋 Procédure de Test

Ce test valide que les changements de couleur d'avatar se propagent automatiquement dans le planning sans rafraîchissement.

1️⃣Préparation

  • ✅ Ouvrir l'application dans un onglet
  • ✅ Se connecter avec votre compte
  • ✅ Aller dans la vue Planning (SemaineVirtualClean)
  • ✅ Noter la couleur actuelle de votre avatar

2️⃣Test de Changement

  • 📱 Ouvrir un nouvel onglet avec l'application
  • ⚙️ Aller dans Paramètres > Couleur de présence
  • 🎨 Changer la couleur pour une couleur différente
  • 💾 Sauvegarder le changement

3️⃣Validation

Retourner sur l'onglet du planning et vérifier :

  • ⏳ L'avatar dans la barre d'utilisateurs change automatiquement
  • ⏳ Les variables CSS sont mises à jour
  • ⏳ Aucun rafraîchissement nécessaire
  • ⏳ Le changement est instantané (< 2 secondes)

4️⃣Test Console

Dans la console développeur, vous devriez voir :

🔄 Préférences mises à jour en temps réel: {presenceColor: "#FF5733"}
🎨 Couleur de présence mise à jour en temps réel: #3b82f6 → #FF5733
✅ Préférences utilisateur chargées
🎨 Variables CSS mises à jour avec la couleur: #FF5733
📢 Réception d'un événement de changement de préférences: {userId: "...", colorChanged: true}
